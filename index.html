¿Qué es?Aporte para salud y riesgos profesionales
Porcentaje (trabajador)3% del salario bruto
Tope mensual$30.00
Base legalLey del ISSS

Ejemplo quincenal (salario mensual $1,200.00 → quincenal $600.00): $600.00 × 3% = $18.00 → tope mensual $30.00, quincenal $15.00

¿Qué es?Aporte para el fondo de pensiones

TopeSin tope

Ejemplo quincenal (salario mensual $408.80 → quincenal $204.40): $204.40 × 7.25% = $14.82
Ejemplo quincenal (salario mensual $1,200.00 → quincenal $600.00): $600.00 × 7.25% = $43.50
📄 ISR (Impuesto sobre la Renta) - Ejemplo quincenal
¿Qué es?Impuesto que se paga al Estado según ingresos
Base exenta mensual$550.00 (Decreto No. 10, abril 2025)
CálculoTabla progresiva sobre salario después de ISSS y AFP
Base legalArt. 26 Ley de Impuesto sobre la Renta
Ejemplo quincenal (salario mensual $408.80 → quincenal $204.40): Base: $204.40 - $6.13 - $14.82 = $183.45 → exento → $0.00
Ejemplo quincenal (salario mensual $1,200.00 → quincenal $600.00): Base: $600.00 - $15.00 - $43.50 = $541.50 → exento (mensual < $550) → $0.00
Ejemplo quincenal (salario mensual $2,000.00 → quincenal $1,000.00): Base: $1,000.00 - $15.00 - $72.50 = $912.50 → ($912.50 - $895.24) × 20% + $60.00 = $63.45 (mensual ISR ≈ $126.90)
